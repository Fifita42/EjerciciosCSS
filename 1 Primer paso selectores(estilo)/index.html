El selector es QUE COSA es lo que queremos cambiar.

Que tipos de selectores existen:
Universal: agregar un *

De tipo: Por el nombre de los elementos, por ejemplo: button, h1, p, etc.

Clases: Agregarlo por clasess por ejemplo, dentro del "p" agregamos class="clasequecreealazar" y en el CSS para llamar a la clase se agrega .nombredelaclase

ID: Dentro de lo que vamosa seleccionar agregamos id="nombreX" y en el CSS agregamos #nombreX
Los Id tienen que ser UNICOS para cada elemento, no se pueden volver a usar para otro elemento.

Por atributo: Dentro de lo que vamos a seleccionar agregamos un atrinuto, por ejemplo ATRIBUTOX="SII" y en el selector de CSS copiamos lo mismo pero entre []

Descendientes: Es para cambiar el contenido que esta dentro de otro contenido, por ejemplo, dentro de un h2 ponemos un p y en el selector de CSS para modificarlo ponemos el contenedor y luego la cosa a cambiar.

Pseudo clases. Esto tambien funciona con las clases, por ejemplo si en la clase class="parrafo25" tenemos un p, en el selector escribimos .parrafo25 p y funcionaria.

Pseudoclasaes: con hover, mientras el mause pase por una linea seleccionada, esta cambiara dependiendo de lo qu eseleccionamos.
