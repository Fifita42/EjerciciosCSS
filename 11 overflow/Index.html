El codigo overflow es la barra del scroll. Tambien permite controlar el contenido que sobresale de una caja.

Su variable por defecto es visible que permite que el texto sobrante sobrepase la caja. despues esta la propiedad:
auto: que genera un scroll en la caja, para leer el texto.

scroll: coloca la barra de scroll por mas que no sea necesaria. overflow-y o overflow-x: en estas podemos agregar las propiedades auto o scroll para colocar el scroll si queremos. Si aparecen las 2 barras, para ocultar una, simplemente ponemos overflow-"x o y":hidden para ocultarla.
